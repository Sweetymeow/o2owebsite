了解O2O商圈

【0手续费】

O2O商圈完全免费，所有交易不收取任何手续费。

【账期极短】

交易完成三个工作日，O2O商圈即会将货款提现至你的银行卡，让你及时回款。（一般提现后1-2个工作日到账）

【支付安全方便】

支持微信支付、信用卡、储蓄卡、支付宝等多种方式付款，且无需开通网银，安全又方便。

O2O商圈主要功能

Previous Next

  • 1、 我的店铺：

    自动定位和修改商家在百度地图上的地理位置坐标，添加修改商家信息和图片包括：商家图片、商户名称、电话、地址、营业时间、商品模板、商家分类等商家基本信息。

  • 2、 分店管理

    连锁商家可以快速添加连锁商家的分店和商品信息，连锁商家的所有分店商品信息在平台内只需要维护一次，所有连锁商家的分店商品信息可以全面共享。

    输入短信验证码，同意协议选中协议前方勾选钮，点击注册，注册成功自动跳到登陆页面。

  • 3、 商家审核

    商家资质通过平台审核后商家就可以正式运营了，一般平台审核需要2-3天完成。

  • 4、 商品管理：

    轻松添加、编辑商品，并能一键分享至微信好友、微信朋友圈、QQ空间和新浪微博。

  • 5、 微信收款：

    不用事先添加商品，和客户谈妥价钱后，即可快速向客户发起收款，促成交易。

O2O商圈君发现有的掌柜的店铺还是空空如也哦好忧伤。是不是有的掌柜还不会添加商品？今天就讲讲如何在商品管理中 添加商品。

发布商品

  • 打开我的页面，点击添加商品进入。

添加商品

  • 点击右下角添加进入添加商品页面。
  • 在这个页面可以维护商品类别（如果没有满意的类别可以点击右上角的添加种类）、添加商品图片、填写商品描述、商品价格、商品库存。
  • 商品图片可以添加多个，第一张上传的图片将作为此商品的头图，商品详情上传的图片都会出现在商品详情介绍中。点击提交，恭喜你！商品添加成功啦！
    •

    注册开店(个人和商家)

  • 您可以将这个商品直接分享去微信好友、微信朋友圈、微博、QQ好友等，也可以继续添加或者预览商品。
  • 商品添加成功后，“商品管理”页面就会显示您刚刚添加的商品啦！
  • 页面中的上架下方“勾选按钮”可控制商品上架下架，点亮勾选按钮就代表该商品在处于上架状态。
  • 反之该商品则处于下架状态，即用户无法浏览或购买该商品。
    •

    第三步 商家认证申请—企业申请

    注册开店(个人和商家) 3

    1. 进入“我的”主页>点击“企业申请”维护企业资质信息提交
    2. 状态变审核中，等待平台的资质审核通过后，这个企业的店铺才能在平台上正式上线运营，平台上的全部买家才能通过买家客户端查看到该企业的店铺和商品，店铺才能够进行交易。
    3. 将以上信息发送到邮箱（kefu@o2oplus.com.cn），标题为“个人转对公业务申请”会有工作人员来处理。
    4. 邮件发送后3-4个工作日帮您处理完成；如果提交的资料不符合，工作人员会回复邮件。

    注册开店(个人和商家) 3

    2、平台上的商户目前主要支持以企业注册，当企业需要在O2O商圈平台上开店并正式上线运营时，或以对公银行账号进行资金提现时，或者您进行O2O商圈的“商家品牌认证”时，您就需要下载“O2O商圈商家版APP”，具体企业认证申请需要提交的资料如下：

    1. 运营人员手持身份证照；
    2. 公司营业执照副本照片；
    3. 法人代表身份证照片和法人本人照片；
    4. 组织机构代码证和开户许可证明照片；
    5. 运营人员个人声明并加盖公司公章，具体内容包括：本公司授权 ______（个人姓名），注册O2O商圈手机号：___________以公司名义开通O2O商圈，并遵守O2O商圈使用协议，任何法律责任由本公司承担，公司名称:__________。公司名称并盖公章、声明日期。

    注册开店(个人和商家) 3

    将以上文件和资料通过手机APP在线上传到平台进行审核，平台审核时限为3-4个工作日，如果平台审核通过，店铺就可以在平台上正式上线运行了，您的手机注册和开店的流程就大功告成了，大家快去开启属于你自己的O2O商圈店铺吧！
